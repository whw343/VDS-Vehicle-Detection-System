## 五、安装过程中的问题与解决方案

### 问题1: PyTorch CUDA 版本下载极慢
- **现象**: 从 PyTorch 官方源下载 torch-2.5.1+cu121（2.45GB），速度仅 98KB/s
- **原因**: 国内访问 PyTorch 官方 CDN 速度极慢
- **解决**: 使用迅雷从 `https://download-r2.pytorch.org/whl/cu121/torch-2.5.1%2Bcu121-cp311-cp311-win_amd64.whl` 下载 whl 文件到本地，再通过 `pip install` 本地安装

### 问题2: 哈希校验失败
- **现象**: `THESE PACKAGES DO NOT MATCH THE HASHES FROM THE REQUIREMENTS FILE`
- **原因**: 下载过程中文件损坏
- **解决**: 重新下载 whl 文件，确保完整性

### 问题3: 清华镜像只有 CPU 版本
- **现象**: 用清华镜像安装的 PyTorch 是 CPU 版本（2.11.0+cpu）
- **原因**: 清华镜像不提供 CUDA 版本的 PyTorch
- **解决**: 本地安装 CUDA 版本的 whl 文件

### 问题4: 依赖版本冲突
- **现象**: torchvision 0.26.0 需要 torch==2.11.0，但安装的是 torch 2.5.1+cu121
- **解决**: 卸载 torchvision/torchaudio，重新从 PyTorch 官方源安装匹配版本
